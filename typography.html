Typography

Our typography is built on "Inter", a carefully crafted typeface designed for computer screens. We use a smaller base font size (14px) for a subtle, elegant, and professional appearance.

Body Paragraphs

.text-lead / 16px (1rem) / Regular

This is the lead paragraph text. It's slightly larger than the base font size and uses a relaxed line height. It's perfect for introductory sections right below page titles.

Body Base / 14px (0.875rem) / Regular

This is the standard body text. It is set at 14px, offering a clean, compact reading layout that conveys a premium density of information without looking cluttered or bulky.

Small Text (.form-text, .form-label) / 12px-13px / Regular or SemiBold
This is small or helper text, typically used for captions, input labels, or subtle hints.
